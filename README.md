# webutils Pdf Links Extractor
A tool To extract PDF Links

## Installation
Clone into the repo and navigate to the webarchive-utils-dev
```console
$ cd webarchive-utils-dev
```

run in the console 
```console
$ npm i
```
For dev Environment 
```console
$ npm run dev
```

nnavigate to

http://localhost:3000/

in the browser
Upload your pdf
